Program Information

This program will make use of command line parameters, input and output files and output manipulation. The program will read in up to 2000 lines from a data file and do further calculations and analysis of the collected data and finally write these results to a file.

The program will read in a space delimited file of integer pairs, where the first int is read as height and second as weight. It will calculate and store the BMI (Body Mass Index) for each height/weight pair. It will then collect the BMIs into different bins according to given intervals. The program will then calculate the mean and standard deviation for the three data arrays (Height, Weight and BMI). Finally, the program creates a histogram of the BMI data and writes this to an output file together with a summary of the statistics. The program is intended to make use of command line arguments taking the input filename and optionally the output filename. The default output file is called: datagram.txt
